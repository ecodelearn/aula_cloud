# Repositório para Aulas de Cloud Computing

Bem-vindo ao repositório `aula_cloud`!

Aqui você encontrará todo o material didático desenvolvido para as aulas de cloud computing, com foco em Python, IA Aplicada, Git/GitHub e Google Cloud Run.

## Conteúdo Principal

O guia completo para a aula pode ser encontrado no arquivo:

*   **[Manual do Aluno: Engenharia de Software e Cloud Deploy](Manual_Aluno_Cloud_Pro.md)**

Neste manual você encontrará:
*   Visão Geral e Arquitetura
*   Engenharia de Requisitos
*   Guia de Git e GitHub
*   Guia de Python com UV (Ambientes Virtuais)
*   Instruções para Implementação, Testes e Deploy no Google Cloud
*   Exemplos de Código (Flask, Python)
*   Desafios Extras

## Estrutura do Projeto

```
aula_cloud/
├── Manual_Aluno_Cloud_Pro.md   # Manual completo da aula (para exportar para PDF)
├── Material_Completo/          # Pasta com arquivos de exemplo e códigos
│   ├── docs/                   # Documentos auxiliares (Git, UV, Dev/Deploy)
│   ├── index.html              # Frontend HTML simples
│   ├── style.css               # Estilos CSS do frontend
│   ├── script.js               # Lógica JS do frontend
│   ├── main.py                 # Código da aplicação Flask
│   ├── requirements.txt        # Dependências Python
│   ├── pyproject.toml          # Configuração UV
│   └── tests/                  # Testes Pytest
└── README.md                   # Este arquivo
```

## Começando

1.  **Clone este repositório:**
    ```bash
    git clone https://github.com/ecodelearn/aula_cloud.git
    cd aula_cloud
    ```
2.  **Siga as instruções do [Manual do Aluno](Manual_Aluno_Cloud_Pro.md) para configurar seu ambiente e fazer o deploy!**
